# Bm Finance (\$BMB)

## Overview

Official frontend repo of [bm.finance](https://app.bm.finance). Easyfarming, easy life.

## Env

Required env vars:

```
REACT_APP_NETWORK_ID = 56 // Network ID of BSC
```

## To Run

```
yarn
yarn start
```

The app will be running at http://localhost:3000/
